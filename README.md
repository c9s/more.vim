========

安裝
=======

    $ make install

用法
=====

- 命令 `:AppendMoreText`
- 快速鍵:  `\`more`

簡單的說，在輸入模式時，輸入 `\`more` 即可隨心所欲自動插入假文。

而輸入 `:AppendMoreText` 命令可在下一行插入假文。
