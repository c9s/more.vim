MORE.vim
========
一用就愛上的中文假文產生器。現在也有 Vim plugin 了！ (rock)

    風雨又調和著節奏，保不定不鬧出事來，也就可以得到百般的快樂。

很困惑嗎？不知道該說些什麼，就呼叫一下 More text 吧！XD

長官勉勵：

    「什麼樣子，生理的作用在一程度以外，鬧出什麼事來？」

安裝
=======

    $ make install

用法
=====

- 命令 `:AppendMoreText`
- 快速鍵:  `\`more`

簡單的說，在輸入模式時，輸入 `\`more` 即可隨心所欲自動插入假文。

而輸入 `:AppendMoreText` 命令可在下一行插入假文。
